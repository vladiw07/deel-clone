# 🌐 Deel – Global Payroll & HR UI Clone

A clean, responsive **Deel-inspired landing page** built with **React.js** and **Tailwind CSS**. This project recreates the design and layout of the Deel.com homepage for educational and portfolio purposes. It features smooth transitions, responsive components, and a modern UI using Tailwind, DaisyUI, and Framer Motion.

> ⚠️ *This is a non-commercial project built for practice and showcasing frontend skills. All rights to the original design belong to [Deel](https://deel.com).*

## ✨ Features

- 🔄 Fully responsive layout (mobile-first)
- 🖱️ Interactive transitions & animations via Framer Motion
- 🧩 Reusable React components
- 🧭 Smooth scroll-based navigation
- 📬 CTA sections (Contact, Demo, Subscribe)
- ✅ Designed with TailwindCSS and DaisyUI for rapid prototyping

---

## 🛠️ Tech Stack

- **React.js** – Component-based architecture
- **Tailwind CSS** – Utility-first CSS framework
- **DaisyUI** – Tailwind component library
- **React Icons** – Icon pack
- **Framer Motion** – Animations & transitions
- **Vite** – For blazing-fast dev server & build
